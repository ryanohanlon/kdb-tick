# Introduction
The purpose of this git repo is to format and document Kx's kdb-tick [repo](https://github.com/KxSystems/kdb-tick)

## Installation

Run on command line

```
q tick.q sym  .  -p 5010	/tick
q tick/r.q :5010 -p 5011	/rdb
q sym            -p 5012	/hdb
q tick/ssl.q sym :5010		/feed
```
